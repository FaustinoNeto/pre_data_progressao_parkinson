# AFT---MINDATA

Um repositório com objetivo avaliativo para a disciplina "Mineração de Dados e Aplicações na Engenharia", ofertada no curso de graduação em Engenharia da Computação. Este repositório contém materiais, implementações, e exemplos práticos relacionados à aplicação de técnicas de mineração de dados em problemas reais da engenharia, sendo utilizado como base para avaliação e desenvolvimento de competências na área.
